# Skrolbak: Drifting Background Component

A Web Component featuring an infinite, drifting, 3D-tiled background with spiral motion, depth/tilt oscillations, and mouse-follow interaction.

## Usage

```html
<script src="animated-background.js"></script>
<drift-bg src="bg.jpg"></drift-bg>
```

## Build

Run `./code/build.py` to copy source to `./released/skrolbak/`.
